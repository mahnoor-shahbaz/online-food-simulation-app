# Online Food Simulation App (Java)

A menu-driven Java console application that simulates a multi-restaurant food ordering system.

## Overview
This project was built to practice programming fundamentals, problem solving and modular program design using Java.

The application allows users to browse restaurant menus, place orders, validate inputs and generate a final bill with discount logic.

## Features
• Multi-restaurant system with separate menus  
• Menu driven console interface  
• Item code validation  
• Quantity validation using loops  
• Modular functions and structured program design  
• Automatic bill generation  
• 10% discount applied on orders above Rs.3000  
• Itemized receipt output  

## Concepts Used
Java fundamentals  
Control structures (if/else, loops)  
Arrays  
Functions and modular programming  
Input validation and error handling  

## How to Run the Project
1. Download or clone the repository
2. Open the project in IntelliJ IDEA
3. Run the main Java file
4. Follow the console menu to place an order
